Alertas Animadas

Las alertas pueden aparecer y desaparecer con animaciones usando JavaScript.

Código para crear una alerta dinámicamente:

const alertPlaceholder = document.getElementById('liveAlertPlaceholder');

const alert = (message, type) => {
  const wrapper = document.createElement('div');
  wrapper.innerHTML = `
    

      ${message}
      
    
`;

  alertPlaceholder.append(wrapper);
}

Sobre las Alertas

Las alertas proporcionan mensajes de retroalimentación para las acciones típicas del usuario.

Características
  • Múltiples variantes
  • Contenido flexible
  • Botones de cierre
  • Animaciones integradas
Las alertas son útiles para mostrar mensajes de éxito, error, advertencia o información.
